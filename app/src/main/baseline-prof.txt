# Baseline Profile for Chat Screen Performance
# This helps ART pre-compile (AOT) critical methods during installation so the
# JIT does not have to warm them up during the first scroll gestures.
#
# IMPORTANT: The live chat surface is the XML RecyclerView path
#   ChatActivity + ChatAdapter (+ ViewHolders) + RecyclerCoordinator + CollageImageView
# The old Compose ChatScreen.kt path is dead code and is intentionally NOT profiled.
#
# Wildcard rules below use the profgen human-readable format:
#   HSPL<package>/**->**(**)**  => mark every method of every class in the package
#                                  as Hot + Startup + Post-startup so it is AOT compiled.
# Unmatched rules are skipped with a warning (they never fail the build).
