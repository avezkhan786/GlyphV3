# Glyph V3
## Client-Facing Product Brochure With Usage Guide

Glyph V3 is a native Android communication platform that combines messaging, media sharing, live calls, privacy controls, AI assistance, social status features, and interactive communication tools in one unified mobile app.

This version of the brochure is designed not only to present the product professionally, but also to explain how users interact with the main features and how those features can be enabled, disabled, or managed inside the app.

## 1. Core Messaging Experience

### One-to-one chat
Glyph supports direct user-to-user conversations with fast real-time delivery, reply support, message states, and a polished chat interface.

**How to use**
- Open the Chats section.
- Select an existing conversation or start a new one from the contact/user list.
- Type a message and send it from the chat composer.

**How to enable or disable**
- This is a core default feature and remains active as part of the main app experience.
- Individual chat behavior can be adjusted from chat settings where available.

### Reply and swipe-to-reply
Users can respond to specific messages to preserve conversation context.

**How to use**
- Long-press or interact with a message to reply.
- Swipe a message in the conversation to trigger quick reply behavior.

**How to enable or disable**
- Reply is part of the standard chat interaction model.
- If needed, gesture behavior can be managed through future UX settings, but in the current product it is part of the core chat flow.

### Typing indicators and expressive typing
Glyph shows live typing activity and extends it with more expressive, animated conversation feedback.

**How to use**
- Start typing in a conversation.
- The other user sees live typing feedback while the message is being composed.

**How to enable or disable**
- Typing activity works automatically during conversation use.
- If a dedicated expressive typing preference is exposed in settings, users can turn that enhanced layer on or off there.

### Disappearing messages
Chats can automatically remove new messages after a selected time period to support privacy and reduce clutter.

**How to use**
- Open the chat.
- Go to contact/chat info.
- Open the disappearing messages option.
- Select a timer such as 24 hours, 7 days, 90 days, or Off.

**How to enable or disable**
- Turn it on by selecting a timer.
- Turn it off by choosing `Off`.

## 2. Media and Attachment Features

### Image, video, and document sharing
Users can send rich content directly in chat for a more complete communication experience.

**How to use**
- Open a conversation.
- Tap the attachment option or camera shortcut.
- Choose image, video, or document.
- Add a caption if needed, then send.

**How to enable or disable**
- Media sharing is available by default.
- Access depends on permission approval such as camera or media access.
- Camera or gallery permissions can be granted or revoked from Android app permissions.

### Voice messages
Users can record and send voice notes with recording feedback and custom playback UI.

**How to use**
- Open a chat.
- Press and hold the voice record control.
- Slide to cancel or lock recording if supported.
- Release or tap send to deliver the recording.

**How to enable or disable**
- Voice recording requires microphone permission.
- Users can allow or deny microphone access from Android settings.

### Video notes
Glyph supports short-form video note communication for more personal, visual messaging.

**How to use**
- Open the media or camera flow from chat.
- Record a short video note.
- Review and send it into the conversation.

**How to enable or disable**
- Requires camera and media permissions where needed.
- Can be controlled through Android permissions or media access availability.

### In-app camera
The app includes its own native capture flow for faster media creation without leaving the product.

**How to use**
- Tap the camera option from the chat screen.
- Capture a photo or video.
- Preview it and send it directly into chat.

**How to enable or disable**
- Requires camera permission.
- Users can enable or disable access from Android permission settings.

### Media compression
Compression tools help balance quality, upload speed, and storage use before sending larger content.

**How to use**
- Select media for sharing.
- Choose an available compression or quality option before sending.

**How to enable or disable**
- This is typically available during the send flow rather than a separate permanent toggle.

## 3. Calling and Live Communication

### Voice calls
Glyph supports live one-to-one voice communication.

**How to use**
- Open a conversation or contact profile.
- Tap the voice call button.
- Accept incoming calls from the full-screen call UI or notification.

**How to enable or disable**
- Calling requires microphone permission and network access.
- Users can decline permissions, or simply not use the feature.

### Video calls
Users can switch from text communication to face-to-face real-time communication.

**How to use**
- Open a chat or contact profile.
- Tap the video call option.
- Use in-call controls for camera, speaker, mute, and end call.

**How to enable or disable**
- Requires both microphone and camera permission.
- Permissions can be granted or removed through Android settings.

### Group calls
Glyph supports multi-party audio and video communication with adaptive layouts and participant management.

**How to use**
- Start a call with multiple participants.
- Join an invited group call.
- Add participants during an active session if permitted by the flow.

**How to enable or disable**
- Group calling is available when call permissions are granted.
- Users can leave the group call at any time using the in-call controls.

### Live audio sharing
This feature supports real-time microphone-based sharing for lightweight live communication.

**How to use**
- Open the live audio feature from the relevant conversation or settings flow.
- Start sharing your microphone.
- Choose or confirm the audience allowed to receive it.

**How to enable or disable**
- Go to Live Audio settings.
- Turn microphone sharing on or off.
- Adjust audience permissions.
- Stop active sharing directly from the feature controls or settings.

### Walkie-talkie mode
Walkie-talkie adds an instant push-to-talk communication model for faster short-form voice interaction.

**How to use**
- Open a supported chat or walkie-talkie entry point.
- Press and hold the push-to-talk control.
- Release after speaking.

**How to enable or disable**
- Open Walkie-Talkie settings.
- Turn auto-accept on or off if desired.
- Adjust the audience or allowed-user scope.
- Microphone permission must be enabled.

## 4. AI and Language Features

### AI assistant
Glyph includes a dedicated AI experience inside the app for smart assistance.

**How to use**
- Open the AI section or AI agent screen from the app.
- Type a request and interact with the assistant conversation.

**How to enable or disable**
- The feature is accessed when needed rather than always active in every screen.
- If AI access should be hidden in future product variants, it can be positioned as a feature toggle at product level.

### AI composer
AI composer helps users improve messages before sending.

**How to use**
- Type a message in chat.
- Open the AI compose actions from the composer.
- Choose actions such as enhance, grammar, tone, or translation.
- Review the result and use the improved text.

**How to enable or disable**
- This is used on demand from the composer rather than permanently running.
- Users can simply avoid opening it when not needed.

### In-chat translation
Users can translate messages inside conversations without leaving the chat experience.

**How to use**
- Select or activate a message for translation.
- Choose the target language.
- View the translated result.
- Play translated audio if available.

**How to enable or disable**
- Translation activates when the user requests it.
- Language selection can be changed from the translation flow.
- It is not forced on all messages by default.

Glyph blends chat and map awareness into the same communication experience.

- Open or expand the map layer.
- Continue chatting or use quick reply while staying in the map context.

**How to enable or disable**

### Map video mode
Map-linked video allows live visual interaction connected to spatial communication.

**How to use**
- Open the interactive map communication layer.
- Turn on live video mode.
- Switch between avatar mode and live video when needed.

**How to enable or disable**
- Video mode can be toggled on or off from the map controls.
- Audio mute and video state can also be adjusted from the same control area.
- Camera and microphone permission are required.

## 7. Privacy and Security Controls

### App lock
App lock adds an authentication layer before the app can be reopened after a timeout.

**How to use**
- Open Settings.
- Go to Security.
- Turn on App Lock.
- Choose biometric or PIN-based protection.

**How to enable or disable**
- Enable from Security settings.
- Disable by turning off App Lock in the same screen.

### Biometric unlock
Users can unlock the app with fingerprint or supported face authentication.

**How to use**
- Open Security settings.
- Select biometric lock type.
- Complete device authentication when prompted.

**How to enable or disable**
- Turn on biometric lock from Security settings.
- Switch to PIN or turn off App Lock to disable it.

### PIN unlock
Users can protect the app with a PIN.

**How to use**
- Open Security settings.
- Choose PIN-based lock.
- Create and confirm the PIN.

**How to enable or disable**
- Turn on PIN lock from Security settings.
- Change PIN or disable App Lock from the same area.

### Locked chats
Sensitive conversations can be separated from normal chat visibility.

**How to use**
- Open the locked chat feature or move supported chats into the protected area.
- Access those chats only through the protected flow.

**How to enable or disable**
- Locked chat behavior is managed from locked chat settings or the protected chat flow.

### Hidden locked chats and secret code
Glyph adds an additional privacy layer by allowing locked chats to be hidden and accessed using a secret code.

**How to use**
- Open the locked chat setup.
- Create a secret code.
- Use the code through the supported search or reveal flow to access hidden chats.

**How to enable or disable**
- Enable by creating a secret code and using hidden locked chat settings.
- Disable by removing the protection flow if the product configuration allows it.

### Blocked contacts
Users can prevent unwanted communication from specific contacts.

**How to use**
- Open the contact or privacy settings area.
- Block the selected user.

**How to enable or disable**
- Block from contact/privacy controls.
- Unblock later from the blocked contacts screen.

### Contact sync
Glyph uses device contacts to help users discover known people already on the platform.

**How to use**
- Allow contact permission when prompted.
- Browse registered users from the contact-based user list.

**How to enable or disable**
- Enable by granting contacts permission.

### Link previews
Shared links can be presented more attractively with preview information.

**How to use**
- Share or paste a supported link.
- Glyph attempts to load preview information automatically.

**How to enable or disable**
- The preview appears when valid link metadata is available.
- Users can ignore or dismiss the preview when presented.

## 10. Reliability and Background Experience

### Push notifications
Glyph keeps users informed about messages, calls, and communication events even when the app is not open.

**How to use**
- Allow notification permission when prompted.
- Receive alerts for supported events.

**How to enable or disable**
- Turn notifications on or off from Android notification settings.
- Fine-grained notification behavior may also be adjustable inside app settings.

### Notification actions
Users can respond faster through notification reply and mark-as-read actions.

**How to use**
- Expand a supported notification.
- Tap reply or mark as read.

**How to enable or disable**
- These actions are available when notifications are enabled.

### Presence and background reliability
Glyph is designed to preserve message reliability, presence visibility, and communication continuity in real-world conditions.

**How to use**
- Use the app normally.
- Accept optional battery optimization guidance if prompted for better background reliability.

**How to enable or disable**
- Reliable background operation improves when battery optimization restrictions are relaxed.
- Users can accept or skip that recommendation.

## What Makes Glyph V3 Distinct

### AI inside the message flow
AI assistance is integrated into composition, which makes it practical and immediately useful instead of feeling detached from real conversation.

### More than standard chat and calling
Live audio, walkie-talkie, live location, map interaction, and map video mode create a broader communication product than a traditional messenger.

### Stronger privacy layers
The combined model of app lock, locked chats, hidden locked chats, and secret code access gives users deeper privacy control than most standard chat apps.

### A more creative status system
Status posting goes beyond simple stories by supporting collage creation, layout editing, text overlays, visual enhancement, and more expressive publishing.

### Native Android product depth
Because Glyph V3 is designed as a native Android app, it is positioned for strong performance, rich permission-aware features, reliable media handling, and deep system integration.
